Możliwości formatowania tekstu w HTML

  • Pogrubienie: < b > ... < / b >
  • Kursywa: < i > ... < / i >
  • Podkreślenie: < u > ... < / u >

  • Przekreślenie: < strike > ... < / strike >
  • Indeks górny: < sup > ... < / sup >
  • Indeks dolny: < sub > ... < / sub >

  • Przejście do nowego wiersza w tekście: < br >
  • Zapobieganie automatycznemu przenoszeniu fragmentu tekstu do nowego wiersza: < nobr > ... < / nobr >
  • Spacja nie łamliwa: & nsbsp
